# mj bijou オンラインストア（Shopify）

mj bijou オンラインストアのテーマコード管理用リポジトリです。

本リポジトリはテーマコードのバージョン管理 および 本番反映の基準ブランチ管理を目的としています。

（※商品・注文データ等のストアデータは含みません）

最終更新日：2026/02/26

## 🔧 GitHub 運用ルール

本プロジェクトは単独運用を前提とし、`main` ブランチを本番コードとします。

※本リポジトリは「main マージ = 本番反映」の運用としていますので、

mainブランチへのマージ前には、必ずローカル確認を実施してください。

```bash
shopify theme dev
```

### 基本フロー

1. GitHub の「Issues」でタスク管理（登録時のイシュー番号を下記 2 で使用）
2. `main` から`作業用ブランチ`を作成

```bash
git checkout -b feature/イシュー番号
```

3. 作業完了後、作業用ブランチからmainに対してプルリクエスト（PR）を作成
4. PRマージ後、ShopifyのGitHub連携テーマに自動反映（＝本番反映）

※main へのマージは「本番リリース」と同義とします。

### ブランチ管理

| ブランチ               | 用途                         | 備考                  |
| ---------------------- | ---------------------------- | --------------------- |
| `main`                 | 本番環境と同期するブランチ   | main で直接作業しない |
| `feature/イシュー番号` | Issue に紐づく作業用ブランチ | 例: feature/10        |

## ⚠️ 注意事項

- Shopify 管理画面側から直接編集した内容は GitHub に自動反映されません。
- 必ず GitHub 経由で管理してください。
